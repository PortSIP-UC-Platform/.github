## What is PortSIP?

PortSIP Solutions Inc. is a global provider of VoIP and Unified Communications (UC) products and SDKs . The company offers a complete portfolio of UC solutions including softphones, PBX systems, video conferencing, and SDKs for developing custom VoIP applications . PortSIP solutions are designed for service providers, enterprises, and system integrators.

Infrastructure teams building communication solutions benefit from PortSIP's flexible deployment options and comprehensive SDK library. System administrators appreciate the Windows-native applications, multi-platform support, and the ability to integrate VoIP into existing applications .

## Installation Guide

### Prerequisites

- Windows 10 or later (64-bit)
- .NET Framework (for SDK development)
- Network connectivity to SIP server

### Option 1: PortSIP UC / PortSIP Softphone for Windows

**Step 1:** Visit the PortSIP website and download the UC client or softphone installer for Windows.

**Step 2:** Run the installer and follow the setup wizard.

**Step 3:** Launch the application .

**Step 4:** Configure the SIP account settings:
- SIP Server (domain or IP address)
- Authentication username and password
- SIP port (default: 5060)
- Transport protocol (UDP, TCP, or TLS)

**Step 5:** Start making calls via the softphone dial pad .

### Option 2: PortSIP PBX

PortSIP PBX is a Windows-native IP PBX that can be installed on Windows Server.

**Step 1:** Download the PortSIP PBX installer for Windows.

**Step 2:** Run the installer as administrator.

**Step 3:** Follow the setup wizard to configure:
- Admin password
- SIP domain
- Network settings

**Step 4:** Access the PBX admin portal via web browser.

**Step 5:** Configure extensions, SIP trunks, and call routing.

### Option 3: PortSIP VoIP SDK

**Step 1:** Download the PortSIP VoIP SDK from the official website.

**Step 2:** Extract the SDK files to your development directory.

**Step 3:** Review the API documentation and sample code.

**Step 4:** Build your custom VoIP application using the SDK libraries.

## Common Configuration

### PortSIP UC / PortSIP Softphone Configuration

| Parameter | Setting |
|-----------|---------|
| Display Name | User Name |
| Username | SIP Extension Number |
| Password | SIP Password |
| Domain | PBX IP Address or FQDN |
| SIP Port | 5060 (or as configured) |
| Transport | UDP / TCP / TLS |
| Registration | Auto-on |

### PortSIP PBX Configuration

| Setting | Default |
|---------|---------|
| SIP Domain | `localhost` |
| SIP Port | 5060 |
| HTTPS Port | 443 |
| Admin Interface | Web-based |
